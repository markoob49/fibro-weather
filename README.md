# ФиброПогода / FibroWeather 🌿🌤️

Персональное веб-приложение для мониторинга фибромиалгии с анализом влияния погодных факторов.

**→ [Открыть приложение](https://markoob49.github.io/fibro-weather/)**

---

## Что делает приложение

- Ведёт дневник симптомов по 8 доменам стандарта FIQR (боль, усталость, скованность, сон, туман, настроение, болезненность, сенсорная чувствительность)
- Автоматически прикрепляет погодные данные к каждой записи (атмосферное давление, Δ давление 12ч, влажность, точка росы, температура, ветер, Kp-индекс, фаза луны)
- Вычисляет флэр-индекс и индекс погодного риска на основе текущих метеоусловий
- Строит прогноз обострений на 48 часов с почасовой разбивкой
- Выявляет персональные погодные триггеры (ранговая корреляция Спирмена с лаговым анализом)
- Генерирует отчёт для ревматолога / рефлексотерапевта с аппендиксом и научными источниками

## Особенности

- **Все данные хранятся локально** на устройстве пользователя (localStorage). На сервер ничего не передаётся.
- Работает как PWA — можно добавить на главный экран смартфона
- Трёхязычный интерфейс: русский, иврит, английский
- Персональный профиль активируется после 20 записей: алгоритм определяет индивидуальные пороги и лаг реакции (через сколько часов после изменения давления нарастает боль)
- Не требует регистрации и установки

## Источники данных

| Данные | Источник |
|---|---|
| Погода (текущая и прогноз) | [Open-Meteo](https://open-meteo.com/) |
| Высота над уровнем моря | [Open-Meteo Elevation](https://open-meteo.com/) |
| Kp-индекс (геомагнитная активность) | [NOAA SWPC](https://www.swpc.noaa.gov/) |
| Геокодирование (прямое и обратное) | [Open-Meteo Geocoding](https://open-meteo.com/) + [BigDataCloud](https://www.bigdatacloud.com/) |

Все API бесплатны и не требуют ключей.

## Уровни доказательности

| Уровень | Параметры | Основные источники |
|---|---|---|
| 🟢 Доказанные | Атм. давление, Δ давление, влажность, точка росы | Fagerlund 2019, Bossema 2013, Strusberg 2002 |
| 🟡 Неоднозначные | Температура, скорость ветра | Neumann 2001, Shutty 1992 |
| 🔵 Экспериментальные | Kp-индекс, фаза луны | Рабочая гипотеза |

## Медицинский дисклеймер

Приложение является персональным инструментом самомониторинга и **не является медицинским устройством**. Не предназначено для постановки диагноза или замены консультации врача. При ухудшении самочувствия обращайтесь к врачу.
